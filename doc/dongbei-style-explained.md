## （一）前言：说整就整，不磨叽

歌词开篇就是一顿絮叨，说什么憋在家里无聊、会编程才能挣钱云云。
这段没语法，但有一个灵魂宣言：

> **"咱得整 Dongbei Style。"**

这就是 dongbei 语言的核心精神——**说人话**。
关键字全是中文，注释可以用东北话写，连变量名都能叫"老王"。
英文字母当然也成，但那多没意思啊。

## （二）你好世界：跟世界唠两句

### 打印语句：`嘀咕` 和 `唠唠`

歌词说：

> 想要发声别的不行"嘀咕"先得说到。
> 说完"嘀咕"第二件事还得要打冒号，
> 要唠的事咱都得放进那双引号。

翻译成人话：**打印语句** 长这样——

```
嘀咕："你好，世界！"。
```

`嘀咕` 是打印关键字，后面跟冒号 `：`，然后是要打印的内容，最后打个句号 `。` 收尾。
`唠唠` 跟 `嘀咕` 是兄弟俩，意思完全一样，随便用哪个。

**例子一：** 问候世界的两种说法，效果一毛一样

```
嘀咕："唉呀妈呀，世界你好！"。
唠唠："唉呀妈呀，世界你好！"。
```

**例子二：** 也可以打印数字和表达式

```
嘀咕：1加2。
嘀咕：三加四。
```

输出：`3` 和 `7`。没毛病。

---

### 句号和感叹号：都行

> 完了打上句号，也是非常重要。
> 要不待见句号，还有那啥，感叹号！
> 跟句号意思一样的，特别适合吐槽！

dongbei 语言用 `。` 或者 `！` 结束语句，两个意思完全一样。
感叹号是给"情绪激动的东北银"准备的。

```
嘀咕："今儿天气不错"。    ← 正常人
嘀咕："今儿天气不错"！    ← 心情特好的东北银
```

---

### 空白换行：爱咋整咋整

> Dongbei 语言空白换行加了都是白加的。

空格、换行、缩进——dongbei 全不在乎，随便加。
下面这俩程序效果完全一样：

```
嘀咕："你好"。嘀咕："世界"。
```

```
嘀咕："你好"。
嘀咕："世界"。
```

---

### 数字：中文阿拉伯都行

> 数字除了阿拉他伯也可以用中文的。

一到十可以直接用汉字写，支持混用：

```
嘀咕：三加4。      ← 3 + 4，输出 7
嘀咕：俩乘以5。    ← 2 * 5，输出 10
嘀咕：十减七。     ← 10 - 7，输出 3
```

特殊情况：`鸭蛋` 和 `零` 都是 0，`俩` 和 `两` 都是 2，`仨` 就是 3。

## （三）活雷锋：变量这玩意儿

### 声明变量：`是活雷锋`

> 烙饼就得裹大葱，dongbei 都是活雷锋。
> 雷锋精神在于装，装成啥样是啥样。

变量就是"活雷锋"——随时准备装东西。声明方式：

```
老王是活雷锋。
```

这就声明了一个叫 `老王` 的变量，初始值是 `啥也不是`（相当于 Python 的 `None`）。
然后用 `装` 给它赋值：

```
老王是活雷锋。
老王装250。
嘀咕：老王。
```

输出：`250`。老王现在是个250了。

**例子：** 把一个变量的值赋给另一个

```
老王是活雷锋。
老张是活雷锋。
老王装666。
老张装老王。
嘀咕：老张。
```

输出：`666`。老张现在跟老王一样是个666。

---

### 加减一：`走走` / `走两步` / `稍稍`

> 装完三，再装五，这个咱叫"走两步"。
> 装完五，再装六，这个操作叫"走走"。
> 好汉能伸也能屈，雷锋"稍稍"就减一。
> "稍一步"，十变九，"稍两步"，九变七。

这几个是给变量加减固定值的快捷操作：

| 关键字 | 意思 |
|--------|------|
| `走走` | `+= 1` |
| `走两步` | `+= 2` |
| `走N步` | `+= N` |
| `稍稍` | `-= 1` |
| `稍一步` | `-= 1` |
| `稍两步` | `-= 2` |
| `稍N步` | `-= N` |

**例子一：** 从 1 数到 5

```
老王是活雷锋。
老王装一。
嘀咕：老王。
老王走走。
嘀咕：老王。
老王走走。
嘀咕：老王。
老王走走。
嘀咕：老王。
老王走走。
嘀咕：老王。
```

输出：`1` `2` `3` `4` `5`。（后面学了磨叽就不用这么费劲了，先忍忍）

**例子二：** 先加仨再减俩

```
老王是活雷锋。
老王装十。
老王走仨步。
老王稍两步。
嘀咕：老王。
```

输出：`11`。加了 3 减了 2，净剩 1。

---

### 四则运算：加减乘除全会

> 光会加减太磕碜，乘除括号也会的。
> 四则运算全掌握，学习雷锋杠杠的。

运算符一览：

| 关键字 | 意思 |
|--------|------|
| `加` | `+` |
| `减` | `-` |
| `乘` / `乘以` | `*` |
| `除以` | `/` |
| `齐整整地除以` | `//`（整除）|
| `刨掉一堆堆` | `%`（取余）|
| `拉饥荒` | 取负（`-x`）|

括号用全角的 `（` 和 `）`。

**例子：** 算个复杂表达式

```
老王是活雷锋。
老王装（三加四）乘以（十减二）。
嘀咕：老王。
```

输出：`56`。`(3+4) * (10-2) = 7 * 8 = 56`，没毛病。

---

### 群众（数组）：`都是活雷锋`

> 一个萝卜一个坑儿，一群雷锋一个村儿。
> 有先来，有后到，村儿里座次很重要。
> 带头大哥叫老大，最后来的叫老幺。
> 老大老二到老幺，每个银都能单挑。

数组叫"群众"，定义时用 `都是活雷锋`（注意不是 `是活雷锋`）：

```
张家庄都是活雷锋。
```

往里加元素用 `来了个`：

```
张家庄来了个"东北"。
张家庄来了个"大米"。
张家庄来了个"猪肉炖粉条"。
```

访问元素：`老大` 是第一个，`老幺` 是最后一个，`老二` `老三` 以此类推：

```
嘀咕：张家庄的老大。     ← 东北
嘀咕：张家庄的老二。     ← 大米
嘀咕：张家庄的老幺。     ← 猪肉炖粉条
嘀咕：张家庄有几个坑。   ← 3（数组长度）
```

**例子：** 装一桌东北菜，报个菜名

```
菜单都是活雷锋。
菜单装「"锅包肉"，"小鸡炖蘑菇"，"猪肉酸菜炖粉条"」。
嘀咕：菜单的老大。
嘀咕：菜单的老幺。
嘀咕：菜单有几个坑。
```

输出：`锅包肉` / `猪肉酸菜炖粉条` / `3`。

## （四）磨叽：循环这东西

### 范围循环：`从...到...磨叽`

> 先说从几到几，然后开始演戏。
> 该唠磕你唠磕，该咋地你咋地。
> 最后磨叽完了，画上点睛之笔。

格式：

```
老王从1到5磨叽：
  嘀咕：老王。
磨叽完了。
```

`老王` 是循环变量，从 `1` 数到 `5`，每次打印一下。
输出：`1` `2` `3` `4` `5`。

**例子：** 打九九乘法表的一行（3的倍数）

```
老王从一到九磨叽：
  嘀咕：三乘以老王。
磨叽完了。
```

输出：`3` `6` `9` `12` `15` `18` `21` `24` `27`。

---

### 遍历群众：`在...磨叽`

> 全村儿挨个儿磨叽，排队注意秩序。

对数组里每个元素操作，用 `在...磨叽`：

```
老张在菜单磨叽：
  嘀咕：老张。
磨叽完了。
```

**例子：** 把菜单每道菜都报出来

```
菜单都是活雷锋。
菜单来了个"酱骨架"。
菜单来了个"地三鲜"。
菜单来了个"东北乱炖"。
老张在菜单磨叽：
  嘀咕：老张。
磨叽完了。
```

输出：`酱骨架` / `地三鲜` / `东北乱炖`，三道菜挨个上。

---

### 无限循环：`从一而终磨叽`

> 磨叽永不放松，那叫从一而终。

```
老王从一而终磨叽：
  嘀咕：老王。
磨叽完了。
```

这是无限循环，老王的值永远是 1（没有步进），一直转到天荒地老。
一般配合 `尥蹶子` 用（见下）。

---

### 跳出和继续：`尥蹶子` / `接着磨叽`

> 要是改了主意，不想接着磨叽。
> 那就尥尥蹶子，dongbei 程序继续。

- `尥蹶子`：相当于 `break`，立刻跳出循环
- `接着磨叽`：相当于 `continue`，跳过本次直接下一轮

**例子：** 从 1 数到 10，碰到 5 就不数了

```
老王从一到十磨叽：
  寻思：老王跟五一样一样的？
  要行咧就尥蹶子。
  嘀咕：老王。
磨叽完了。
```

输出：`1` `2` `3` `4`。到 5 就尥蹶子了，5 本身也没打印出来。

**例子：** 从 1 到 10，跳过偶数

```
老王从一到十磨叽：
  寻思：老王刨掉一堆堆二跟零一样一样的？
  要行咧就接着磨叽。
  嘀咕：老王。
磨叽完了。
```

输出：`1` `3` `5` `7` `9`。偶数全被跳过，只剩单数。

## （五）套路：函数这门手艺

### 定义套路：`咋整` / `整完了`

> 要练体操先压腿儿，想学套路先起名儿。
> 写完名字写"咋整"，爱做的事儿卯劲儿抡。
> 最后写上"整完了"了，套路就算你练成了。

套路（函数）的写法：

```
报数咋整：
  嘀咕："我是东北银！"。
整完了。
```

`报数` 是套路名字，`咋整：` 开始定义体，`整完了。` 表示定义结束。

---

### 调用套路：`整`

> 使用套路很简单啊，名字写在"整"字儿后啊。

```
整报数。
```

**例子：** 定义并调用一个打招呼的套路

```
打招呼咋整：
  嘀咕："东北你好，我来了！"。
整完了。

整打招呼。
整打招呼。
```

输出两遍：`东北你好，我来了！`。套路就是这么好用，写一次到处整。

---

### 带参数的套路

> 有的套路带参数啊，括号里头杵一杵啊。

参数放在括号里，套路里可以直接用：

```
问好（称呼）咋整：
  嘀咕："你好啊，"、称呼、"！"。
整完了。

整问好（"铁子"）。
整问好（"大姐"）。
```

输出：`你好啊，铁子！` 和 `你好啊，大姐！`。

---

### 返回值：`滚犊子吧`

套路可以给调用方返回一个结果，用 `滚犊子吧`（相当于 `return`）：

```
平方（那啥）咋整：
  滚犊子吧 那啥乘以那啥。
整完了。

嘀咕：整平方（四）。
嘀咕：整平方（七）。
```

输出：`16` 和 `49`。`滚犊子吧` 后面跟着的值就是返回值。

---

### 递归套路：自己调自己

> 有的套路爱递归呀，自个儿名字推一推呀。

套路可以在自己体内调自己，这就是递归。经典例子：阶乘。

```
阶乘（那啥）咋整：
  寻思：那啥比一还小？
  要行咧就滚犊子吧一。
  滚犊子吧那啥乘整阶乘（那啥减一）。
整完了。

嘀咕：整阶乘（五）。
```

输出：`120`。5 的阶乘 = 5×4×3×2×1 = 120，算得贼准。

---

### 嵌套套路：套路套套路

> 有的套路套套路啊，这里的山路十八弯啊。

套路里头可以调别的套路，随便套：

```
加一（那啥）咋整：
  滚犊子吧那啥加一。
整完了。

加两个一（那啥）咋整：
  滚犊子吧整加一（整加一（那啥））。
整完了。

嘀咕：整加两个一（十）。
```

输出：`12`。调了两层 `加一`，10 变成 12，没毛病。
